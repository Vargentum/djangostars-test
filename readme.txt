Верстка написана на адаптированном под мои нужны фреймворке Fireshell.
Какие технологии применены в проекте:
html5, css3, sass, compass, jquery, grunt, микроформаты.

Файлы которые которые идут в продакшн находятся в папке /app
Исходники в /src
Файлы .scss разбиты следующим образом:
- переменные
- сущности которые не меняются от проекта к проекту (классы-хелперы и т.п.)
- сущности которые меняются (шрифты, импорты плагинов и т.п.)
- основной файл стилей.

Шрифты и тексты могут не совпадать, т.к. верстать пришлось не из psd-макета, а из обычной картинки.
Я решил не тратить время на подгонку шрифтов и текстов, т.к. в реальном проекте есть макеты, с которых можно взять всю необходимую инфо.

Решил улучшить несколько мелких деталей с помощью css3 анимаций - работают при ховере логотипа и серых кнопок со стрелками.

Время выполнения - 5 часов.
